### Global对象
    Global对象是系统中的固有对象，全局对象，
    固有对象就是系统已经实例化完毕的对象，只可以使用不可以创建
    
    作用：将系统中的函数全部添加到当前对象当中成为当前对象的成员方法
          将系统中定义的变量添加到当前对象中成为当前对象的成员属性
    
    区分函数和方法？
        如果方法属于Window对象（Global对象），通常称之为函数
        如果方法不属于Window对象，属于其他对象，通常称之为方法

    区分属性和变量

### Date对象
    该对象是js中的时间对象
    2016/5/5  3:5:12
    5/15/2016
    猴年马月 子时...
    
    在js中为了方便时间的存储和计算，引入了时间戳的概念
    时间戳也叫作Unix时间戳，他是用从1970年1月1日0时0分0秒0毫秒到指定时间的毫秒数，js时间戳可以使用285616年....
    
    1天=24小时
    1小时=60分钟
    1分钟=60秒
    1秒=1000毫秒
    
    1.实例化一个当前时间对象
        var now = new Date();
    2.实例化一个指定时间对象 
        传入指定的时间戳
        var 变量 = new Date（时间戳）；
        传入时间格式的字符串
        var 变量 = new Date（时间格式字符串）
        例如：1970/1/1   1970-1-1   1970.1.1   5:12:23
        分别传入年月日时分秒参数
        var 变量 = new Date（年，月，日，时，分，秒，毫秒）；

### Array对象原型
    什么是数组？
    数组就是一组数据
    在js中没有数组这种数据类型，数组是对象创建的
    
    下标：用于区分数组中不同数数值的标志就是下标
    键：下标就是键
    值：每个键对应的数据就是值
    键值对：键+值 就是键值对
    元素：键值对就是元素
    
    键分为两种：
    以数字作为下标的键，这种数组称之为索引数组
    以字符串作为小标的键，这种数组称之为关联数组
    注意：在js中只有索引数组，y没有关联数组，这个和其他语言不太一样
    必须是从0开始连续的索引数组！

### 数组操作

    var 变量 = 【】；创建一个空数组
    var 变量 = 【值，值。。。】；创建具有指定元素的数组
    var 变量 = 【整数】；创建一个指定整数的数组，并不是长度！

    }

#### 二维和多维数组
    一维数组：
    如果一个元素中的所有值都不是数组，那么这个数组就是一个一维数组
    
    二维数组：
    如果一个一维数组的元素中包含一维数组，那么这个数组就是二维数组
    
    三维数组：
    如果一个一维数组的元素中包含二维数组，那么这个数组就是三维数组
    
    ......
    多维数组：
    如果一个数组的维度超过三维，统称为多维数组
    
#### 多维数组的访问操作：

#### 数组相关的函数
    concat() 连接数组元素
    ·  格式：新变量 = 数组变量.concat(值，值...)
    ·  参数值，可以是正常数值也可以是数组类型，都可以连接
    
    join（） 使用字符将数组元素连接成一个字符串
    ·  格式：新变量 = 数组变量.join（指定字符串）
    
    pop（）在数组的结尾处弹出一个元素（直接改变原有数组）
    ·  格式：结果变量 = 数组变量.pop（）；
    ·  注意：该方法直接操作时原有数组，返回值为弹出的元素
    
    push（）在数组的结尾处添加元素（直接改变原有数组）
    ·  格式：结果变量 = 数组变量.push（值，值。。）
    ·  注意：该方法直接操作原有数组，返回值为添加后的数组长度
    
    shift（）在数组的开头移除一个元素（直接改变原有数组）
    ·  格式：结果变量 = 数组变量.shift（）；
    
    unshift()在数组的开头难添加属性元素（直接改变原有数组）
    ·  格式：结果变量 = 数组变量.unshift(值，值，，，）
    
    reverse（）数组倒转法
    ·  格式：结果变量 = 数组变量.reverse()

    ·  格式：结果变量 = 数组变量.slice(开始下标【，结束下标】）

    ·      结果变量 = 数组变量.splice(指定位置，删除个数，新增元素）
    注意：该方法非常适用于js游戏，需要保证数组的连续性
