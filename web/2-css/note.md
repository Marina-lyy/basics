# CSS
- 什么是css？
- 层叠样式表。Cascading Style Sheets
- 作用就是用来修饰HTML标签
## 如何在页面中使用CSS
#### 1.外链式css
    <link href="css文件地址" type="text/css" rel="stylesheet" />
    案例v01
#### 2.导入式css
    <style>
        @import url('css文件地址');
    </style>
    案例v02
#### 3.嵌入式css
    <style>
        书写CSS代码
    </style>
    案例v03
#### 4.内联式css
    <标签style=“css属性”>
     注意：导入式和嵌入式如何共存
     如果style标签中即存在导入式css，又存在嵌入式css，那么导入式css的@import语法必须在第一行出现，而且后面需添加分号
     案例v04
#### 4种css方式的选取规则：
    特征：
    优点：外链式css和导入式css最大优点是能够实现代码标签和样式属性的完全分离
    缺点：需要加载两个文件，可能导致在网速较慢的情况下，页面开始没有样式，需要在css文件加载完毕时才能显示所有样式
    嵌入式
    优点：嵌入式的优点是能够保证在html加载完毕的情况下，css一定加载完毕，保证页面自始至终结构不乱
    缺点：该方法会导致文件过大，需要较长的时间才能完全显示页面
    内联式（不允许使用该方式设置css）
    优点：不需要进行标签的选取操作，为javascript提供的操作css方法
    缺点：他的优先级最高，所设置之后很难替换
      推荐方式：
        1.单独使用外链式或者导入式，更加有利于SEO的优化（个人小型站点适用）
        2.单独使用嵌入式一般情况下常见大型新闻类网站
        3.混合使用嵌入式和导入、链接式，将页面的结构css样式写入嵌入式部分，细节样式写入导入、外链式css。
     
## css的书写规范
    选择器{
        css属性名：css属性值；
        css属性名：css属性值；
        css属性名：css属性值；
        ...
    }
    选择器：用于选取指定类型的标签，分为很多种
    css属性名：用于设置需要修改的样式名称
    css属性值：用于设置属性结果类型
    
## css的选择器分类
    1.html选择器、元素选择器
    直接使用HTML标签的名称作为选择器，就可以直接选中页面中同名的html标签，统一添加样式
    案例v05
    
    2.类选择器、class选择器
    通过为制定的标签添加class属性，并且定义一个属性值，在选取时使用，+自定义class值局可以选中所有具有当前class值的表
    案例v06
    
    3.id选择器
    通过为指定的标签添加id属性并且定义一个id值。（该值在一个页面中不可以出现重复），在选取元素时使用#+id值就可以选中具有id值的那个元素
    注意：id绝对不可以重复，但是对后期的js有很大的影响
    案例v07
    
    4.组和选择器
    如果多个选择器使用相同的css样式，那么我们可以进行css精简操作，使得css属性只书写一遍即可
    
    选择器，选择器...{
                共用的css样式
        }
    案例v08
        
    5.关系选择器
    关系选择是利用标签之间的嵌套关系进行元素的选取，可以很好的限制选取元素的范围
    
    选择器1 选择器2{
            css样式
    }
    注释：选中选择器1元素内部所有符合选择器2的元素，无论有几层关系
    v09
    
    6.通用选择器
      选中所有元素，使用时一般用于浏览器之间的样式重置操作或者配合关系选择器进行限定选取
    清除浏览器默认样式：
        *{margin:0;padding:0;list-style:none;...}
    限定选取元素
    ul *{
            css样式
    }
    案例v10
    
    7.伪元素选择器
    伪元素选择器选中的不是元素本身，而是元素的某一个状态或者一部分内容。
    a标签的设置：
        a:link  设置正常连接的颜色
        a:hover  设置鼠标经过时的样式
        a:active  设置鼠标点击时的样式
        a:visited  设置链接访问过后的状态
    其他标签的设置：
        tr:hover  设置单元格行标签鼠标经过的样式
        其他标签可以使用鼠标经过样式hover，但是不能使用a标签中其他的状态
        选择器：first-letter  选中元素内部第一个字符进行样式设置
        选择器：first-line  选中元素内部第一行进行样式设置
        选择器：after  选中元素内容的后面的部分进行样式或者内容的添加
        选择器：before  选中元素的内容的前面部分进行样式或者内容的添加
    案例v11
        
## CSS属性介绍
#### 背景属性
    background-color属性  设置元素的背景颜色
    取值：颜色值
    注意：可以使用颜色单词，RGB模式或者hex模式，如果红色，绿色，蓝色值中每个颜色的两个数字都相同，可以缩写成三位数字，例如#ff9900-》#f90
    background-image属性  设置元素的背景图片
    格式：background-image：url（图片地址）
    
    background-repeat属性  设置背景图片的重复方式
    repeat   横向纵向重复（默认值）
    no-repeat   不重复
    repeat-x    横向重复
    repeat-y    纵向重复
    
    
    background-position属性  设置背景图片的位置
    格式；background-position:值1 值2
        当前背景图片离左侧的距离为值1，离顶部的距离为值2
    格式：background-position：值1
        当前背景图片离其中一个距离为值1，另一个居中对齐，（如果值为百分比或者像素，那么值1则表示水平方向）
    取值：1.长度单位  像素。。。
          2.使用百分比
          3.使用位置单词  left左  right右 top上  bottom下  center 居中
      
    background-position-x属性  设置背景图片离左侧的距离
    background-position-y属性  设置背景图片离顶部的距离
    案例v13
    
    background-attachement属性  设置背景图片的绑定方式
    值：
        scroll  背景图片随页面内容滚动（默认值）
        fixed   背景图片不随内容滚动
    注意：当前css2的属性设置值scroll和fixed仅适用body标签
    案例v14
    
    background属性  背景设置的综合属性
    值：可以将其他所有背景属性的值在此设置，并且没有顺序关系
    案例v15
    
#### 尺寸属性
    width属性  设置元素的宽度
    值：长度单位
    
    height属性  设置元素的高度
    取值：长度单位
    
    min-width属性  设置元素的最小宽度
    取值：长度单位
    注意：最小宽度设置之后，如果外层元素的宽度大于最小宽度设置，则当前元素使用外层元素的最大宽度
          如果外层元素的宽度小于当前元素的最小宽度设置，那么元素的宽度为设置的最小宽度值
    该属性在页面布局中使用较少。
    min-height属性  设置元素的最小高度
    取值：长度单位
    注意：如果元素中的内容没有超过最小高度值，当前元素高度为最小高度值
          如果元素中的内容高度超过最小高度值，那么当前元素高度会增高，使得内容完全包含在元素当中
    该属性在论坛的帖子和回复中经常使用。
    
    max-width属性  设置元素的最大宽度
    取值：长度单位
    注意：如果外层元素宽度大于当前最大宽度值，那么当前元素的宽度就是最大宽度值，
          如果外层元素宽度小于最大宽度值，那么当前元素的宽度会随之变小。
      
    max-height属性  设置元素的最大高度
    取值：长度单位
    注意：如果内容高度小于最大高度值，高度由内容的高度决定
          如果内容高度大于最大高度值，高度就是最大高度值，内容会溢出
- 案例v16
          
    
#### 字体属性
    font-size属性  设置字体的大小
    取值：长度单位
    
    font-family属性  设置字体的类型
    格式：font-family:'字体1','字体2'...
    注意：字体必须是来自用于的操作系统中存在的字体
    多个字体之间是并列关系，系统会选取最前面的支持的字体进行显示，如果不支持则顺序向后查找支持的字体
    
    font-weight属性  设置字体是否是粗体
    取值：
        normal  正常字体
        bold   粗体
        lighter  比外层元素更加细（正常字体）
        bolder  比外层元素更粗（粗体）
        100-500   正常字体
        600-900   粗体
        
        800是常用的粗体代词
        
    font-style属性  设置字体是否是斜体
    取值：
        normal  正常字体
        italic  使用斜体显示
        oblique  使用正常字体，但是浏览器将正常字体变斜显示（不常用）
        
    font-variant属性  设置字体是否变为小型大写字母
    取值：
        normal  正常字体  默认值
        small-caps  小型大写字母
    注意：只对英文有效
    
    font属性  字体设置综合属性
    格式：风格  小型大写字母  粗体  大小/行高  字体类型（必须按照顺序书写）
    注意：当前属性有个严格的顺序和数量要求，所以不推荐使用

#### 文本属性
    text-align属性  设置内容的水平对齐方式
    值：
        left   靠左对齐（默认）
        center  水平居中
        right  靠右对齐
    注意：对块状元素有效
   
    vertical-align属性  设置内容的垂直对齐方式
    取值：
        top  顶部对齐
        middle  垂直居中
        bottom  底部对齐
    注意：仅对内敛元素和表格的单元格元素有效
    案例v19
    
    line-height属性  设置行高（一行的高度）
    取值：长度单位
    注意：经常讲行高设置为和元素等高，实现单行文本的垂直居中效果。
    案例v20
    
    text-transform属性  设置英文的大小写转换方式
    取值：
        none  不做修改（默认值）
        capitalize   首字母大写
        uppercase   全部变为大写字母
        lowercase   全部变为小写字母
        
    letter-spacing属性  设置字母之间的距离
    取值：长度单位
    注意：汉字作为字符处理，该属性对其生效。
    
    word-spacing属性  设置单词之间的距离
    取值：长度单位
    案例v22
    
    text-indent属性  设置元素内部第一行和左侧的缩进距离
    取值：长度单位
    注意：在进行汉语相关的内容布局时，通常使用em作为单位，在p标签中比较常见
    案例v23
    
    white-space属性  设置元素对于空白的处理方法
    值：
        normal  正常处理方式（默认值）
        pre  显示源代码中的格式，遇到边界不换行
        nowrap  强制内容不换行（回车），除非遇到br标签
        pre-wrap  显示源代码中的格式，遇到边界换行
        pre-line  保持换行字符显示，但是空白字符不显示
    案例：v23

### 表格属性
    table-layout属性  设置表格布局中宽度的设定方式（布局算法）
    值：auto  根据内容多少来分配表格的宽度
        fixed  根据实际情况来分配宽度（不根据内容，而是根据设定）
        
    border-collapse属性  设置表格中的单元格是单独的合并
    值；separate  单元格独立  -》 每个单元格有自己独立的边框
        collapse  单元格合并  -》  所有边框单元格共享
    注意：常常用于设置表格的所有边框为1像素。
          虽然可以通过table标签的cellspacing=0来设置边框合并，但是实际结果得到的是2像素的边框
          
    border-spacing属性  设置独立单元格的间距
    取值：长度单位
    格式1：  border-spacing：值
             单元格之间横向和纵向的间距都是制定的值
    格式2：  border-spacing：值1 值2：
             单元格之间的横向间距是值1，纵向间距是值2
    注意：IE67不支持该属性，可以通过表格自带的cellspacing属性来实现类似的效果（不能单独设置水平和垂直间距，效果类似于格式1）
    
    caption-side属性  设置表格描述标题的位置
    值： top  描述在表格的顶部
         bottom  描述在表格的底部
         
    注意：IE67不支持该属性对caption的设置，可以再caption标签中添加valign属性进行设定为top和bottom也可以实现该效果。
    
    empty-cells属性  设置空格单元格的显示方式
    值： hide  隐藏空单元格
         show  显示空单元格
    注意：在IE67以下浏览器默认值为hide，其他浏览器默认值为show；
    
### 定位属性
    postion属性  设置单元格的定位方式
    值：
        static  静态定位方式（默认值）
            元素默认的定位方式就是静态定位，就像一块瓷砖用水泥固定在地上
        案例v30
        absolute  绝对定位方式
            绝对定位元素会使得当前元素脱离文档流（瓷砖不在教室的范围之内），可以在任何范围内移动，而且不会因为移动影响页面中的其他元素，就像一块瓷砖在楼顶。
            1.如果当前绝对定位元素的外层元素都是静态定位元素，只能以html标签作为定位参考点。
            2.距离当前绝对定位元素外层最近的一个非静态定位元素作为参考点。
        案例v31 ，v32
        relative  相对定位方式
            相对应为元素，没有脱离文档流，还在页面当中，可以移动，就像一款地板上的瓷砖松动了，可以移动。
            行对定位的参考点是当前元素原来的位置（static时的位置）
        案例33
        fixed  绑定定位方式
            长在你的眼睛上一块瓷砖，眼睛怎么动都无所谓，反正这块瓷砖在你眼神中永远固定位置。
            
            left属性  设置定位元素距离定为参考点左侧的距离
            top属性  设置定位元素距离定位参考点顶部的距离
            right属性  设置定位元素距离定位参考点右部的距离
            bottom属性  设置定位元素距离定为参考点底部的距离
        案例 v34，v35
            
        z-index属性
            设置定位元素的层级关系，所有元素z-index的值是0；
            取值；整数值，没有上限
            在相同定位的情况下，谁的z-index值越大，显示越靠上！
            注意：静态定位元素无法使用该属性
        案例v36

### 布局属性
    display属性  设置元素的显示方式
    值：  none  不显示元素也不占用物理空间
          inline  显示为行内元素
          block  显示为块元素
          
          inline-block  显示为行内块装元素
          行内块状元素特征：即可以设置宽高，也可以在一行共存
          
          list-item  显示为li列表标签的样式
          table  显示为table标签的样式，做为块元素显示
          inline-table  显示table标签的样式，但是作为内敛元素显示
          table-row   显示为tr标签的样式
          table-cell  显示td标签的样式
    注意：display仅改变元素的显示效果，对用于的视觉改变，不会改变标签的意义，也不可以使用显示效果对应标签都而元素
    案例：v37， v38
    
    visibility属性  设置元素显示与否
    值：
        inherit  继承（默认值）  显示与否由父元素的显示决定
        visible  显示
        hidden  隐藏元素  占用物理空间，需要注意和display：none的区别
        collapse  专门用于表格行和单元格的隐藏设置属性。  IE6以下不支持
    案例v39
    
    overflow-y属性  设置元素垂直方向溢出的处理方式
    值：
        auto  如果内容超过元素，则出现滚动条，不超出则不出现滚动条
        scroll  一直存在滚动条，但是有没有滑块取决于内容的高度
        hidden  对于超出高度的内容进行隐藏处理
        visible  对于超出的内容做显示处理（默认值）
    案例v40
    
    overflow-x属性  设置元素水平方向溢出的处理方式
    值：
        auto  如果内容超过元素，则出现滚动条，不超出则不出现滚动条
        scroll  一直存在滚动条，但是有没有滑块取决于内容的宽度
        hidden  对于超出宽度的内容进行隐藏处理
        visible  对于超出的内容做显示处理（默认值）    
    注意：正常情况下内容超过元素水平宽度会自动换行到下一行.
          不会换行的情况下:
          1.如果内层包含的不是文字，而是其他图片，元素等内容，而且这些内容宽度超过外层元素
          2.连续的英文字符，没有空格和标点符号，会被浏览器当做一个完整的单词显示，不会强制换行
          3.设置内容显示效果为white-space:nowrap.强制不换行属性
    案例v41
    overflow属性  设置元素内容溢出（同时设置水平和垂直方向）的处理方式
    值：
        auto  如果内容超过元素，则出现滚动条，不超出则不出现滚动条
        scroll  一直存在滚动条，但是有没有滑块取决于内容的宽度、高度
        hidden  对于超出宽度、高度的内容进行隐藏处理.
        visible  对于超出的内容做显示处理（默认值）
    案例v42
    
    float属性  设置元素的浮动方式
    值：  none  元素不浮动（默认值）
          left  左浮动
          right  右浮动
    注意：浮动属性和定位属性是不同的属性
    
    浮动元素的特征是从元素原有的位置漂浮起来，可以进行左右漂浮，但是不能设置移动位置
    浮动元素不会脱离文档流，位置处于静态定位元素和决定定位之间
    
    clear属性  清除浮动属性
    值：
        none  不清除浮动（默认值）
        left  清除左浮动
        right  清除右浮动
        both  清除左右浮动（常用）
    注意：浮动和清除浮动属性是在页面布局中最常用的属性    
    案例：v43
