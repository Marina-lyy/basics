# Css颜色，文本字体
- css颜色表示法
    - 1. 颜色名表示，比如：red红色，gold金色
    - 2. 16进制数值表示，比如：#000表示红色，这种可以简写成 #00f
    - 3. RGB颜色：红（R）、绿（G）、蓝（Ｂ）、透明度（A）background.rgba(0,0,0,0.5);
    - 4. RGBA颜色：红（R）、绿（G）、蓝（B）、透明度（A）backguound.color:rgba(0,0,0.5)
